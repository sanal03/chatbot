Q: Is Groq API free?
A: Yes! Free tier available with generous limits.

Q: Can I change the AI model?
A: Yes! Easy to switch between Mixtral, Llama 3.1, Llama 2, Gemma.

Q: How do I deploy this?
A: Backend to Railway, frontend to Vercel. See README.md for details.

Q: Can I add a database?
A: Yes! Easy to integrate MongoDB, PostgreSQL, etc.

Q: Is this secure?
A: Yes for dev. For production, see CONFIG_GUIDE.js security section.

Q: Can I customize the knowledge base?
A: Yes! Edit SYSTEM_PROMPT in server.js.

Q: How do I change colors?
A: Edit styles.css - search for #667eea or #764ba2.

Q: What if I get an API error?
A: Check SETUP.md troubleshooting section or API_GUIDE.md.
